# Understanding the UI

![Battle Interface Overview](https://picsum.photos/800/400?random=10)
*The comprehensive Broken Arrow battle interface*

Mastering Broken Arrow's interface is crucial for effective battlefield command. This guide breaks down every element of the battle interface and how to use them efficiently.

## Main Menu Overview

![Main Menu Navigation](https://picsum.photos/700/300?random=11)
*Navigate the main menu efficiently*

Before diving into battle, familiarize yourself with the main menu:

### Four Main Tabs
1. **Home Tab** - Main dashboard and news
2. **Arsenal Tab** - Unit browser and battlegroup creation
3. **Multiplayer Tab** - Join or create lobbies
4. **Editor Tab** - *(Locked during beta)*

### Key Interface Elements
- **Profile Section** - Click to view statistics and customize your profile
- **Chat Button** - Access team communication
- **Settings** - Game parameters and controls
- **Screenshot Mode** - Hide HUD by clicking this, reactivate via Broken Arrow logo

## Battle Interface Layout

![Command Interface](https://picsum.photos/900/350?random=12)
*Real-time command and control interface*

Once in battle, you'll encounter a sophisticated command interface:

### 1. Reinforcement Menu
**Location:** Left side of screen  
**Purpose:** Deploy units onto the battlefield

![Reinforcement Panel](https://picsum.photos/300/500?random=13)
*Unit deployment and reinforcement controls*

- **Unit Categories:** Infantry, Vehicles, Aircraft, Support
- **Point System:** Each unit costs points from your battlegroup
- **Transport Options:** Infantry can be paired with transport vehicles
- **Supply Integration:** Add supplies to cargo vehicles

### 2. Match Timer & Objectives
**Location:** Top center  
**Purpose:** Track victory conditions

- **45-minute time limit** for standard matches
- **Objective indicators** show capture status
- **Victory achieved** by controlling all objectives or majority at time expiration

### 3. Communication Markers
**Location:** Above minimap  
**Purpose:** Coordinate with teammates

![Communication Tools](https://picsum.photos/400/200?random=14)
*Team coordination and communication interface*

Four marker types available:
- **Warning** (F1) - Alert teammates to threats
- **Attack** (F2) - Coordinate offensive operations  
- **Defend** (F3) - Request defensive support
- **Supplies** (F4) - Mark supply/logistics needs

**Usage Tips:**
- **5 markers maximum** active at once
- **1-minute duration** before auto-removal
- **Right-click** to remove markers
- **Drag and drop** to reposition

### 4. Minimap
**Location:** Bottom right  
**Purpose:** Strategic overview

![Tactical Minimap](https://picsum.photos/300/300?random=15)
*Strategic battlefield overview*

- **Terrain overview** with elevation indicators
- **Unit positions** for friendly forces
- **Objective locations** clearly marked
- **Enemy contacts** when detected

## Unit Interface Elements

### Selection Panel
**Location:** Bottom center  
**Purpose:** Manage selected units

![Unit Selection](https://picsum.photos/600/150?random=16)
*Unit selection and management interface*

- **Unit thumbnails** for all selected forces
- **Individual targeting** - Click thumbnail or press TAB to command single unit
- **Group management** - Maintain selection while giving individual orders

### Status Panel
**Location:** Bottom right  
**Purpose:** Detailed unit information

Displays comprehensive data for the currently selected unit:
- **Unit specifications** and capabilities
- **Current health** and ammunition status
- **Equipment loadout** and customizations
- **Combat effectiveness** ratings

### Action Panel
**Location:** Bottom center/right  
**Purpose:** Issue commands

![Command Panel](https://picsum.photos/400/250?random=17)
*Unit command and action interface*

Contains context-sensitive buttons for:
- **Movement orders** and formations
- **Combat commands** and targeting
- **Special abilities** unique to unit type
- **Fire missions** for artillery units

## Unit Label System

![Unit Labels](https://picsum.photos/600/200?random=18)
*Understanding unit status indicators*

Understanding unit labels is crucial for battlefield awareness:

### Standard Label Elements
1. **Unit Name** - Appears when hovering or selecting
2. **Cover Indicator** - Forest/building concealment status
3. **Return Fire Indicator** - Shows defensive posture
4. **Resupply/Repair Icons** - Current maintenance operations
5. **Health Bar** - Green to red damage indication
6. **Progress Bars** - Aiming, reloading, or other actions
7. **Radar Icon** - Active radar systems (green)
8. **Critical Effects** - Equipment damage (yellow/red)

### Critical Damage Indicators
**Four types of critical effects:**
- **Targeting** - Reduced accuracy and increased aim time
- **Detection** - Decreased sensor capability
- **Reloading** - Slower ammunition handling
- **Mobility** - Reduced movement speed or immobilization

## Interface Shortcuts and Tips

![Hotkey Reference](https://picsum.photos/500/300?random=21)
*Essential keyboard shortcuts and controls*

### Essential Hotkeys
- **F1-F4** - Communication markers
- **TAB** - Cycle through selected units
- **X** - Deploy smoke grenades/screens
- **P** - Precision strike for aircraft
- **Alt** - Line of sight visualization tool

### Efficiency Tips
- **Pin unit statistics** when comparing equipment options
- **Use compact mode** for quick unit overview during battles
- **Master marker placement** for effective team coordination
- **Learn to read critical effects** at a glance

## Advanced Interface Features

### Line of Sight Tool
- **Hold Alt** to display visibility lines
- **Identify blind spots** and concealment opportunities
- **Plan movement routes** using terrain

### Screenshot Capabilities
- **Hide all UI elements** for clean screenshots
- **Reactivate** by clicking the Broken Arrow logo
- **Useful for** documentation and sharing
